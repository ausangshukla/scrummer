scrummer
========

Simple Scrum Tool

Goals
========
1. The simplest scrum tool to develop, deploy & use

Installation
========
1. Clone the git repo
2. bundle
3. rake db:create
4. rake db:schema:load
5. rake scrummer:generateFakeAll

Customize
========
All customizations should be placed in .env
